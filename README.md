## How to (UNIX/MacOS)

```bash
# once cloned, navigate to repo
cd /path/to/monolog

# allow running the script
chmod +x ./run.sh

# run the script with or without the debug flag
# flag will run the celery/django servers with debugpy so you can use the debugger configs
./run.sh [--debug]
```
